# Landing Page para Academia Fitness
O projeto tem como objetivo divulgar os serviços prestados e auxiliar na prospecção de clientes. 


# Responsividade
Para melhor usabilidade foram utilizados recursos de responsividade e assim a landing page funciona em vários dispositivos.
Em relação a largura de tela para foram considerados:
- Celulares -até 425px;
- Tablets - até 768px
- Computadores - até 1024px
- A landing page também funciona em telas maiores que 1024px

# Tecnologias utilizadas
* HTML5
* CSS3
* Git e GitHub
